# Notes 3
## What is a graphical user interface (GUI)?
set of programs that allows a user to interact with computer system via icons

## What is a desktop environment?
Implementation of the desktop metaphor made of bundle program running on top of a computer operating system

## What is the command line interface (CLI)?
The user interacts with computer by inputting commands.

## How do I access the command line interface (CLI)?
Linux Console and Terminal Emulator

## What is a virtual console?
is a terminal session that runs in linux system memory 

## What is a terminal emulator?
a program that allows you to access the linux CLI when using the GUI

## What is bash?
is the shell that is interpreting your commands

## What is the shell prompt?
appear whenever the shell is ready to accept input
