## How to install and remove software using the APT command
how to install: sudo apt install 
example: sudo apt install cuyo

How to remove: sudo apt remove
example: sudo apt remove cuyo

## How to create a shell script step by step including screenshots and how to run it. Try to be as detailed as possible.

### Step 1 Create the file
First go to text editor do #!bin/bash and then click on hamburger tab and save as example.sh in a folder that you made named scripts 
### Step 3 Add your code
first line will be #!bin/bash so go to the second line and write any code. for example echo "hello world" echo -n "DATE: " date . then proceed to save 
### Run the Script
Go to the terminal and write down bash ~/scripts/example.sh    then it will show the code you made from text editor
