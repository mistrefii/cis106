# Notes 6

## mkdir
* used for creating a single directory or multiple directories
* create multiple directories, separate each directory name with a space
### Formula
* `mkdir` + `the name of the directory`
### Examples
* `mkdir wallpapers` `mkdir wallpapers/forest`

## touch
* is ued to creating files
### Formula
* `touch`
### Examples
* `touch list` `touch Downloads/games2.txt`

## rm
* remove files
* default does not removes directories.
### Formula
* `rm`
* `rmdir`
* `rm -r + directory name`
### Examples
* `rm list`
* `rmdir Downloads/games`
* `rm -r Downloads/games`

## cp
* copies files/directories from a source to a destination
* command has many options
### Formula
* `cp + files to copy + destination`
### Examples
* `cp Downloads/wallpapers.zip Pictures/`
* `cp -r ~/Downloads/wallpapers ~/Pictures/`

## mv
* moves and renames directories
* where source is the file or directory that you want to move and destination is where the directory or file is going
### Formula
* `mv + source + destination`
### Examples
* `mv Downloads/homework.pdf Documents/`
* `sudo mv ~/Downloads/theme /usr/share/themes`
